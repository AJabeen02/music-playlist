Music Playlist Management System

This project is a simple program developed in C for managing a music playlist efficiently.

✨ Features
➕ Add songs to the playlist
📃 Display all songs
🔍 Search for a song
❌ Delete songs from playlist
✏️ Update song details
💻 Language Used
C Programming
📌 Description

The Music Playlist Management System allows users to manage their favorite songs using basic operations like adding, viewing, searching, updating, and deleting records. It is a console-based application designed for learning data handling in C.
